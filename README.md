## ⚡ the moment everything changed

</div>

> I built **DiagnoseAI** — a voice-first constrained LLM for medical symptom assessment.
> During testing, it returned a response that was **structurally perfect** and **clinically dangerous**.
> I couldn't explain why.
>
> That single failure redirected everything.
> I stopped asking *"does the model work?"*
> I started asking *"why does it fail — and what does it take to actually fix that?"*
>
> Every project since is an answer to that question.

**[→ GitHub](https://github.com/PriyankaAsthana/RAKT_Blood_Bank_Management)**

Found a **race condition** in existing blood bank systems — units in transit could be concurrently reallocated. Built a time-bound token reservation mechanism to eliminate it.
